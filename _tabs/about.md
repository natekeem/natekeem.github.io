# About

이 블로그는 Project AI Autoblog를 만들면서 남기는 개발 기록입니다.

처음 목표는 AI를 활용해 영어권 40대 이상 독자를 대상으로 한 수익형 블로그 네트워크를 실험하는 것이었습니다. 하지만 바로 자동화부터 만들기보다, 먼저 수동 발행과 검수 체계를 통해 콘텐츠 품질과 운영 흐름을 검증하기로 했습니다.

여기에는 다음 내용을 기록합니다.

- 프로젝트 구조와 의사결정
- Codex, Roo Code, Claude Code, ChatGPT 사용 경험
- Jekyll/Chirpy 기반 블로그 구축 과정
- SEO와 AdSense 리스크를 학습한 내용
- 자동화하기 전에 수동으로 검증한 과정
- 실패, 막힘, 방향 전환

이 블로그의 글은 AI의 도움을 받아 작성할 수 있지만, 최종 발행 전에는 사람이 검토합니다.
