# Privacy Policy

`나태킴의 커밋로그`는 현재 별도의 회원가입, 댓글 시스템, 문의 양식을 운영하지 않습니다.

현재 상태 기준으로 이 사이트는 의도적으로 개인 정보를 수집하지 않습니다.

## 현재 사용하지 않는 것

현재 이 블로그에는 다음 기능을 아직 사용하지 않습니다.

- Google Analytics
- GoatCounter
- 댓글 시스템
- AdSense
- 제휴 링크 추적
- 문의 폼

향후 조회수 분석, 댓글, 광고, 제휴 링크 등이 추가되면 이 페이지를 업데이트하겠습니다.

## GitHub Pages

이 사이트는 GitHub Pages를 통해 호스팅됩니다.  
GitHub Pages는 브라우저, 네트워크 환경 등에 따라 기본적인 접속 로그 또는 기술 정보를 처리할 수 있습니다.

이 문서는 법률 자문이 아니며, 현재 블로그 운영 상태를 투명하게 설명하기 위한 문서입니다.
