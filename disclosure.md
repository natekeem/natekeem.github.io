# Disclosure

`나태킴의 커밋로그`는 Project AI Autoblog를 만들어 가는 과정을 기록하는 개발 로그입니다.

## AI 사용에 대해

이 블로그의 글은 ChatGPT, Codex, Roo Code, Claude Code 같은 AI 도구의 도움을 받아 작성하거나 다듬을 수 있습니다.

다만 최종 발행 전에는 사람이 내용을 검토합니다.

## 광고와 제휴 링크

현재 이 블로그에는 다음 항목이 없습니다.

- AdSense 광고
- 유료 협찬
- 제휴 링크
- 제품 제공 리뷰

향후 광고, 협찬, 제휴 링크가 추가되면 해당 글 또는 이 페이지에 명확히 표시하겠습니다.

## 수익 주장에 대해

이 블로그는 AI-assisted blogging, 자동화, SEO, GitHub Pages 실험 과정을 기록합니다.  
수익을 과장하거나 검증되지 않은 결과를 성공 사례처럼 포장하지 않겠습니다.
